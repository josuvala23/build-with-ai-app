Diferencia técnica
API de Base de Datos

Una API de base de datos se utiliza para almacenar, consultar y modificar información estructurada. El servidor envía consultas y recibe datos exactos.

API de IA

Una API de IA como Gemini procesa lenguaje natural y genera respuestas dinámicas usando modelos de lenguaje. El servidor debe preparar prompts y manejar respuestas generativas.

Seguridad

Exponer una API Key en el frontend es peligroso porque cualquier usuario puede verla desde el navegador y utilizarla de forma maliciosa.

El backend protege la API Key porque:

mantiene las credenciales ocultas
controla las solicitudes
valida usuarios
evita abuso de la API
Google AI Studio

Google AI Studio es una plataforma de Google para probar modelos Gemini, crear prompts y obtener claves API para integrar inteligencia artificial en aplicaciones.
